# 📈 NovaTrade

> A full-stack MERN portfolio project inspired by Zerodha.

---

## 📖 Overview

NovaTrade is a full-stack trading platform built to demonstrate modern web development concepts using the MERN stack.

This project focuses on implementing real-world backend and frontend architecture, authentication, REST APIs, database integration, CRUD operations, and responsive user interfaces.

**Disclaimer:** NovaTrade is a fictional trading platform created solely for educational and portfolio purposes. It does not provide real financial or investment services.

# 📌 Project Status

This project is intended to showcase full-stack web development concepts rather than serve as a production-ready trading platform.

### Implemented

- JWT Authentication
- Login & Signup
- Protected Routes
- CRUD Operations
- REST APIs
- MongoDB Integration
- Database Relationships
- Modern Dashboard
- API Integration

### Partially Implemented

Some sections such as Holdings and Positions currently use demonstration data and are not fully dynamic.

The goal of this project was to demonstrate the implementation of core MERN concepts instead of recreating every feature of a production trading platform.

# 📱 Responsiveness

Basic responsiveness has been implemented throughout the application.

Building a pixel-perfect responsive interface was not the primary objective of this project. The main focus was understanding full-stack architecture, backend development, authentication, APIs, and database design.
